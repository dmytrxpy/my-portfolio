Graphic Designer Portfolio
A professional, bilingual (EN/DE) portfolio gallery designed for visual storytelling, built for high performance and seamless deployment on Vercel.

✨ Core Features
Bilingual Support: Minimalist localization (English & German) powered by Zustand.
Data-Driven: All project data and content are centralized in projects.json.
Fluid UI: High-end aesthetics with smooth transitions using Framer Motion.
Optimized Assets: Dynamic media loading through the public/ directory for maximum efficiency.
🛠️ Technology Stack
Core: React + TypeScript
Build System: Vite
Styling: Tailwind CSS
Animations: Framer Motion
State: Zustand
🚀 Setup & Execution
To run this project locally, ensure you have Node.js installed, then follow these steps:

Install dependencies:

npm install
Start the development server:

npm run dev
Build for production:

npm run build
